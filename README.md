# 📖 Steve WordMem Pro | 史蒂夫单词辅助记忆系统

这是一个轻量化、纯前端的单词记忆与测试工具。旨在通过简洁的交互，帮助用户高效录入、记忆并自我检测单词掌握情况。

## ✨ 核心功能

* **灵活录入**：支持“单词:释义”格式批量导入，并能通过特定符号（如双空格或 `..`）实现释义换行。
* **记忆卡片**：沉浸式翻页卡片设计，支持点击翻面查看释义。
* **多选题模式**：自动从词库中抽取干扰项，练习识别能力。
* **拼写挑战**：手动输入单词，强化拼写记忆。


* **进度反馈**：实时统计积分，每达到 100 分将触发彩虹纸屑庆祝动画。
* **本地存储**：自动保存学习进度与积分，刷新网页不丢失数据。

## 🚀 快速开始

1. **下载项目**：保存 `index.html` 到本地。
2. **直接运行**：用任何现代浏览器（Chrome, Edge, Safari）直接双击打开该文件即可使用。
3. **配置词库**：在“词库”选项卡中按格式输入内容，点击“保存并更新”开始学习。

## 🛠️ 技术栈

* **HTML5 / CSS3**：采用 CSS 变量与 Flexbox 布局，适配移动端。
* **JavaScript (Vanilla JS)**：纯原生脚本，无需任何外部库，响应极速。
* **Local Storage**：利用浏览器本地缓存存储积分数据。

## 📝 录入规范

在词库配置页中，请遵循以下格式：

```text
单词:释义内容
Focus:v. 聚焦  n. 中心
System:n. 系统，体系

```

* `:` 用于分隔单词与解释。
* `      ` (双空格) 或 `..` 会被解析为换行，方便查看多重含义。
